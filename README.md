# Zoo Match System

Система предназначена для зоопарков и позводяет хранить информацию о животных, добавлять, удалять их, создавать заявки на составление пары с другими животными (сайт знакомств для животных в зоопарке). Хранит информации об имени животного, возрасте, поле, заболеваниях, содержит отметку о готовности к составлению пары. Сотрудники могут отправлять и принимать заявки, оставлять сообщения по заявкам. Система реализована с помошью Spring Boot и React, в качестве бд использован PostgreSQL.
